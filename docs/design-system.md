# Design System (MVP)

## Backgrounds

- Primary background: `bg-slate-950`
- Card/Table background (default): `bg-slate-900`
- Card/Table background (alternate): `bg-slate-800`

## Accent colors

- Primary accent: `amber-400`
- Secondary accent: `sky-400`
- Danger/Rare/Epic accent: `orange-500`

## Text colors

- Primary text: `text-slate-100`
- Secondary text: `text-slate-400`
- Muted text: `text-slate-500`

## Buttons

### Primary button

- Background: `bg-amber-400`
- Hover: `hover:bg-amber-300`
- Text: `text-slate-950`

### Secondary button

- Background: `bg-sky-500`
- Hover: `hover:bg-sky-400`
- Text: `text-white`
