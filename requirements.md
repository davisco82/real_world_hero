# Project requirements (local)

## Runtime and framework

- PHP `8.3`
- Laravel `12.x`
- Livewire `3.x`
- Filament `3.x`

## Frontend

- Vite
- Tailwind CSS

## Infrastructure

- Docker Compose
- Database: MySQL and PostgreSQL (switchable via `.env`)
- Redis
- Queues (Redis-backed Laravel queue)
- Horizon

- App URL: `http://localhost:8111`
- Vite URL: `http://localhost:5173`
- MySQL: `127.0.0.1:3307`
- Postgres: `127.0.0.1:5433`
- Redis: `127.0.0.1:6380`
